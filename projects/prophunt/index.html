Overview

Prop Hunt is a comprehensive Flutter-based mobile application designed to connect property owners (landlords) with potential tenants in Nigeria. The app provides a seamless platform for property listing, discovery, and rental management with a focus on user experience and functionality.

Key Features & Functionality

For Landlords:

  • Property Listing : Add and manage property listings with detailed information
  • Tenant Management : Track and manage current and prospective tenants
  • Property Analytics : Monitor property performance and rental status
  • Verification System : Property verification for increased trust
  • Payment Integration : Handle rental payments and financial transactions

For Tenants:

  • Property Discovery : Browse and search through available properties with advanced filtering
  • Saved Properties : Bookmark favourite properties for later viewing
  • Property Details : View comprehensive property information including images, amenities, and pricing
  • Rental Management : Track current rentals and rental history
  • Profile Management : Complete user profiles with verification system

Tech Stack

  • Flutter
  • Riverpod for state management
